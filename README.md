💻 GalaxyTours
Este é um projeto desenvolvido para a atividade formativa de Front-End, focado arrumar uma interatividade para um App desenhado para telas grandes.

🛠️ Tecnologias Utilizadas
HTML5 (Semântica e Acessibilidade)
CSS3 (Estilização e Feedback Visual)
JavaScript (Manipulação de DOM e Eventos)
📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado: / ├── index.html └── src/ └── img/ ├── scripts/ ├── styles/

🚀 Funcionalidades Implementadas
Adaptado o site para diferentes dispositivos. Foi implementado um ajuste de layout para tablets e smartphones utilizando a classe
.grid-pacotes e um ponto de quebra em 480px.
No modo mobile, os elementos passam a se organizar verticalmente com texto centralizado, garantindo melhor visualização.
Além disso, foi aplicada interatividade nos botões,
que escurecem ao passar o mouse, proporcionando uma melhor experiência ao usuário.
